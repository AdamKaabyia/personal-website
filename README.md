# Adam Kaabyia - Personal Website

Personal portfolio and CV website showcasing my projects, skills, and experience as a Software Engineer at Red Hat.

## Live Site

[adamkaabyia.github.io/personal-website](https://adamkaabyia.github.io/personal-website/)

## Features

- Responsive design with dark/light mode support
- Animated hero section with typewriter effect
- Project portfolio with filtering
- Smooth scroll navigation
- Lazy-loaded images

## Tech Stack

- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)
- Vanilla JavaScript
- GitHub Pages (deployment)

## Local Development

Simply open `index.html` in a browser, or use a local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`

## Deployment

Automatically deployed to GitHub Pages on push to `main` branch via GitHub Actions.
